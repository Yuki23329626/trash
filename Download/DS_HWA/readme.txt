GENERAL USAGE NOTES
-------------------------------------------------------------------------

 - First, you need to execute the file "mazeg.out". You may need the
   competence to execute this file in Linux-like systems.
   You can type in "chmod +x mazeg.out" in command line of csie workstation
   and get the competence to execute it.

 - If you don't execute the file "mazeg.out" to produce a file "maze.txt" 
   first, it would return an error message or just produce a wrong answer.

 - The answer will be write into "result.txt". You can check the answer
   in that file.
